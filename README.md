# 日次タスク管理システム

## 📋 アプリ概要

**日次タスク管理システム**は、チームの日々の業務を効率的に管理するためのWebアプリケーションです。WBSを日次レベルに落とし込み、出勤メンバーの時間軸ベースでタスクを管理できます。

### 🌟 主な特徴

- **🏢 チーム管理**: 出勤者を選択してチーム全体の作業を可視化
- **📱 レスポンシブ対応**: PC・スマートフォンどちらでも快適に利用可能
- **⏰ 時間管理**: 作業時間の記録と集計で生産性を向上
- **📊 進捗管理**: タスクの状態管理（未開始・進行中・完了・保留）
- **🔒 セキュア**: Supabase認証による安全なユーザー管理
- **📈 履歴確認**: 過去の稼働実績を月別で確認可能

## 🚀 アクセス方法

**本番環境URL**: https://task-manager-akihiro-arais-projects.vercel.app

## 👥 対象ユーザー

- **管理者・マネージャー**: チーム全体のタスク進捗を管理したい方
- **チームメンバー**: 自分のタスクを効率的に管理したい方
- **プロジェクトリーダー**: 作業時間の把握と改善を図りたい方

## 🔧 操作方法

### 1️⃣ アカウント登録・ログイン

1. **アクセス**: 上記URLにアクセス
2. **新規登録**: 「アカウントをお持ちでない方はこちら - 新規登録」をクリック
3. **情報入力**: メールアドレスとパスワードを入力
4. **登録完了**: アカウント作成後、自動的にダッシュボードに移動

### 2️⃣ スタッフ管理（PC・スマートフォン共通）

#### スタッフの追加
1. **出勤者選択**エリアを展開
2. **「+ スタッフを追加」**をクリック
3. **スタッフ名を入力**して「追加」ボタンをクリック

#### 出勤者選択
1. **出勤者選択**エリアで今日出勤するメンバーにチェック
2. 選択された人数がリアルタイムで表示されます

### 3️⃣ タスク管理

#### PC版での操作
1. **日付選択**: 上部の日付を変更してタスクを管理したい日を選択
2. **タスク作成**: 
   - スタッフ名を選択
   - タスク名を入力
   - 開始時間・終了時間を設定
   - WBSコード（任意）を入力
   - 「タスクを追加」をクリック
3. **進捗更新**: 各タスクの状態ボタンをクリックして進捗を更新
   - 🔴 未開始
   - 🟡 進行中  
   - 🟢 完了
   - ⏸️ 保留

#### スマートフォン版での操作
1. **個人ビュー**: ログインユーザーのタスクのみ表示
2. **タスク作成**: 「+ タスクを追加」から自分のタスクを作成
3. **簡潔な表示**: 画面サイズに最適化された見やすいレイアウト

### 4️⃣ 稼働履歴の確認

1. **「稼働履歴」タブ**をクリック
2. **月を選択**して過去の実績を確認
3. **詳細確認**: 
   - 日別の作業時間
   - タスク完了率
   - 月間統計
4. **🔄 更新**ボタンで最新情報に更新

### 5️⃣ 便利な機能

#### 自動計算機能
- **稼働時間**: 開始・終了時間から自動計算
- **完了率**: タスクの進捗状況を自動集計
- **月間統計**: 平均作業時間や総タスク数を表示

#### リアルタイム更新
- **自動更新**: 最新の情報を常に表示
- **手動更新**: 更新ボタンでいつでも最新情報を取得

- **稼働時間**: 実際にタスクに費やした時間
- **完了率**: 完了したタスクの割合

## セットアップ手順

### 1. 環境構築

```bash
# 依存関係インストール
npm install
```

### 2. Supabase設定

1. [Supabase](https://supabase.com)でプロジェクトを作成
2. SQL Editorで`sql/init.sql`を実行してテーブル作成
3. `.env.local`ファイルを作成：

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. 開発サーバー起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## データベース構造

### users テーブル
- Supabase Authと連携したユーザー情報

### staff_members テーブル
- スタッフ情報（名前、アクティブ状態）

### tasks テーブル
- タスク情報（日付、担当者、タスク名、時間、ステータス等）

## 使い方

### 1. アカウント作成・ログイン
- 初回利用時はアカウントを作成
- メール・パスワードでログイン

### 2. スタッフ登録
- サイドバーの「出勤者選択」から「+ スタッフを追加」
- スタッフ名を入力して登録

### 3. 出勤者選択
- その日の出勤者をチェックボックスで選択
- 選択されたスタッフのみテーブルに表示

### 4. タスク管理
- **PC**: テーブルのセルをクリックしてタスク追加・編集
- **モバイル**: カード形式でタスク管理
- タスクをクリックしてステータス変更

### 5. リアルタイム同期
- 複数人での同時編集が自動的に同期
- 他のユーザーの変更がリアルタイムで反映

## デプロイ

### Vercel自動デプロイ
1. GitHubリポジトリをVercelに接続
2. 環境変数を設定：
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. 自動デプロイが実行される
